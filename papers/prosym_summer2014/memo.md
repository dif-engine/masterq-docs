## 1

```
夏のプログラミング・シンポジウム2014 ご発表者各位：

幹事の、明星大学の丸山です。
お世話になっております。

シンポジウムから時間が空いてしまい、失礼いたしました。
表記の通り、報告集論文についてご案内いたします。

1. 概要
夏のプロシンでは報告集を作成し、来年1月に開催されます
冬のプロシンで、
- 今年の夏プロの報告集
- 来年の冬プロの予稿集
を合わせて発行いたします。

従いまして、冬のプログラミング・シンポジウムと体裁を
合わせて作成することとなります。一般的な投稿論文とは
やや異なり、
- B5サイズ
- ヘッダ・フッタ等なし
- ページ番号なし
という体裁となります。

最終的に「全てのフォントを埋め込んだPDFファイル」
として作成していただき、ご提出をお願いいたします。

　　　** 締切は 10/31(金) です **

2. 提出物の一覧
以下3種類のファイル及びデータをご提出いただきます。

(1) 論文本体のPDFファイル
詳細は3.で述べます。

(2) 必要事項を記入済みの著作権譲渡契約書（のPDFファイル）
下記よりダウンロードできるPDFをご利用ください。
印刷したものに記入してスキャンするか、PDFに直接
入力なさっても結構です。
http://www.ipsj.or.jp/copyright/ronbun/joto.html

(3) 電子図書館登録用メタデータ
論文本体に含まれる情報の一部（タイトル、概要、キーワード等）
を、入力フォームに記入する形でご提出ください。

3. 論文テンプレート
冬プロの報告集担当幹事である、松崎＠高知工科大さんが作成
してくださった論文サンプルを本メールに添付しましたので、
ご確認ください。

また、体裁の参考資料として、LaTeX及びWordのテンプレート
となるファイルを下記にご用意しました。いずれも、A4サイズの
PDFの中に、B5サイズの版面が仕上がる前提で作成しております。
http://mlab.is.meisei-u.ac.jp/~kazutaka/spro2014/

いずれの場合も、全てのフォントが埋め込まれたPDFファイルを
作成して、ご提出くださいますようお願いいたします。

3-1. LaTeXの場合
松崎＠高知工科大さんが作成してくださった、夏プロ用スタイル
ファイルをご利用ください。上記URLの spro_report.tar.gz
に関連するファイルが含まれています。

スタイルファイル等は iso-2022-jp でエンコードされている
ので、ご自身の環境に合わせて文字コードは適宜変換してください。

3-2. Wordの場合
丸山が、情報処理学会のWordテンプレート(*)を元に、LaTeXの
スタイルファイルと同様の変更を加えたファイルが spro-word.docx
です。
(*) http://www.ipsj.or.jp/journal/submit/style.html

Wordをご利用の場合、以下の点に注意してファイルを作成してください。

- 最終的な仕上がりは、LaTeX版PDFにならうこと
-- ヘッダ・フッタなし、ページ番号なし
- 細かい体裁は情処の研究報告用テンプレートを参照
- 余白は以下の通り変更すること
-- 上：40mm
-- 下：43mm
-- 左右：29mm

4. 提出方法
「2. 提出物の一覧」で述べた3点をアップロードないし記入して
いただくため、Webフォームを作成して後日ご案内いたします
（10月中旬頃を予定しております）。

準備が整いましたら改めてご連絡いたします。

5. 提出期限
10月31日(金) が提出期限です。

ご案内は以上です。

ご不明な点、不都合などありましたら、幹事団のメールアドレス
s-prosym2014@ipsj.or.jp までお問い合わせください。

よろしくお願いいたします。
```

## 2

```
夏のプログラミング・シンポジウム2014 発表者の皆様：

明星大学の丸山です。
お世話になっております。

報告集論文の体裁について、重要な情報が漏れておりました。
論文のページ数は **6〜12ページ** の範囲でご執筆ください。

今回のテンプレートはB5 + 10ptとなっており、一般的な
A4サイズで執筆するよりはページ数が増えることになると
思われます。

また、今回は「ビューティフル・インタフェース」という
テーマですので、スクリーンショット等の図を多用する
必要があり、ページ数が足りないというケースもあるかと
思います。原則として上限は12ページですが、どうしても
超えてしまうという場合は出版担当と調整いたしますので、
幹事団 s-prosym2014@ipsj.or.jp までご相談ください。

以上です。
よろしくお願いいたします。
```
